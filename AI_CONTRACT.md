# AI RULES — READ FIRST

This file defines mandatory rules for AI coding tools.
All instructions in this file are authoritative for this repository.

# REPOSITORY: openbotstack-docs

## ROLE:
This repository documents the philosophy, architecture, and usage
of OpenBotStack.

## IT SHOULD:
- Explain WHY design decisions exist
- Describe system boundaries and trade-offs
- Provide diagrams and examples
- Help contributors understand the mental model

## IT MUST NOT:
- Contain implementation-specific hacks
- Drift from actual system behavior

## DOCUMENTATION STYLE:
- Precise, opinionated, and explicit
- Prefer rationale over marketing language
